Introducción

En el campo de la informática y los sistemas, el empleo de una herramienta como GIT y GITHUB será de gran ayuda para los desarrolladores, con la ayuda del instructor, logramos comprender el uso de esta herramienta, para que sirve y como crear un repositorio para en un futuro poder programar, crear y compartir con otros desarrolladores y trabajar en línea bajo un mismo proyecto. 

            
OBJETIVO


La intención a la que se  quiere llegar con esta actividad es que como futuros ingenieros de sistemas vamos adentrando en el mundo 
del desarrollo de sitios web,pero además conocer de herramientas útiles, que nos permitirán como desarrolladores de sitios web  
trabajar de una manera mas eficaz desde cualquier lugar con acceso a internet en  cualquiera de nuestros proyectos sincronizado en 
GIT y GITHUB.
