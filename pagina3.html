ACTIVIDAD 2. CREAR EL GUIÓN Y LA MAQUETACIÓN PARA UN OVI

Construir sitios web, por medio de un sistema de control de versiones como GIT con lenguajes de marca en HTML5 y CSS3, para potenciar las habilidades en el desarrollo de metodologías y herramientas especializadas en línea. 
   
            
 
Objetivo específico 1


 
La intención a la que se  quiere llegar con esta actividad es que como futuros ingenieros 
de sistemas vamos adentrando en el mundo del desarrollo de sitios web,pero además conocer 
de herramientas útiles, que nos permitirán como desarrolladores de sitios web  trabajar de
una manera mas eficaz desde cualquier lugar con acceso a internet en  cualquiera de nuestros
proyectos sincronizado en GIT y GITHUB.
